ms

Delay before search triggers. Lower = faster response, higher = fewer API calls.
Recommended: 0ms (instant) for best UX. Testing: Try 200ms vs 500ms vs 1000ms.

chars

Minimum characters before search starts. Lower = more suggestions, higher = more precise.

items

Maximum number of dropdown results. More = scrolling required, fewer = might miss items.

Changes apply immediately to autocomplete dropdowns
