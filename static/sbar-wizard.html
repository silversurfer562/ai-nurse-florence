SBAR Wizard Help

Quick Start: Creating an SBAR Report

  1. Navigate through the 4 steps using the Next button
  2. Fill in required fields (marked with *)
  3. Use AI features to enhance your documentation
  4. Click "Complete" to generate your SBAR report
  5. Copy, print, or save your report for handoff

Using AI Text Enhancement

  1. Write your notes in informal language (e.g., "pt has chest pain")
  2. Click the purple "Enhance with AI" button next to the field
  3. Review the AI-generated professional text
  4. Edit as needed - you have full control
  5. The AI learns from SBAR best practices for each section

Checking Drug Interactions

  1. Enter current medications in the Background step
  2. Click the red "Check Interactions" button
  3. AI parses medication names from your text
  4. Review top 3 interactions with severity levels
  5. Major interactions are flagged with warnings
  6. Use full Drug Checker for complete analysis

Getting Priority Suggestions

  1. Complete vital signs and assessment in Step 3
  2. Click "Suggest Priority" button
  3. AI analyzes vital signs and clinical findings
  4. Review suggested priority (STAT, URGENT, or ROUTINE)
  5. Read the AI's reasoning for the recommendation
  6. Make final decision based on clinical judgment

SBAR Best Practices

  1. Situation: Be concise - state the problem in 1-2 sentences
  2. Background: Include relevant history, not entire chart
  3. Assessment: Use objective data - vitals, labs, observations
  4. Recommendation: Be specific about what you need
  5. Use professional, clear language throughout
  6. Include only pertinent information for handoff

Saving and Exporting Your Report

  1. Save Draft (Browser): Saves to browser storage - resume work later on this computer only
  2. Download as .txt: Downloads plain text file to your computer for permanent storage
  3. Copy to Clipboard: Copy report text to paste directly into EMR or documentation system
  4. Print: Opens print dialog for physical hard copy
  5. Auto-save: Your work is automatically saved to browser every 30 seconds while editing

Tips & Tricks

  1. Tooltips: Hover over any button to see what it does
  2. Quick Save: Use the floating save button (bottom right corner)
  3. Navigation: Required fields must be filled to proceed to next step
  4. File Format: Reports are saved as plain text (.txt) for universal compatibility
  5. Privacy: Draft saves are local to your browser - not sent to server
